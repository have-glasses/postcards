# 贵州特色食品数智化技术创新与应用研究中心

这是一个基于 Next.js + Tailwind CSS 的团队官网展示页，用于介绍“贵州特色食品数智化技术创新与应用研究中心 · 食品加工研发团队”的品牌信息、成员构成、成果展示与联系方式。

## 当前页面内容

- 顶部悬浮导航
- Hero 首屏标题（已采用书法题字风格）
- 团队概览指标
- 核心成员展示区
- 论文 / 专利 / 项目成果展示区
- 联系方式区
- 品牌页脚

## 页面已配置的视觉风格

- 深色科技感背景
- 玻璃拟态卡片
- 顶部固定导航
- 滚动入场动画
- 中文题字风格标题字体
- 适合品牌展示的高对比排版

## 当前文案结构

- 顶部品牌文案：`贵州特色食品数智化技术创新与应用研究中心`
- 首屏主标题：`食品加工研发团队`
- 首屏说明：突出“数智赋能特色食品研发”与“产业合作对接”

## 成果内容

页面中的成果区已更新为更贴近当前团队方向的内容，包含：

- 论文成果
- 项目成果
- 案例价值

并支持继续在 `app/page.tsx` 中直接维护对应数据。

## 成员字段支持

当前成员卡片已支持展示：

- 姓名
- 职责角色
- 单位
- 电话
- 邮箱
- 证书
- 研究方向
- 个人简介
- 标签
- 个人照片

## 技术栈

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## 项目结构

- `app/layout.tsx`：全局布局与页面元信息
- `app/page.tsx`：页面主体内容
- `app/globals.css`：全局样式、背景与字体风格

## 本地运行

安装依赖：

```bash
npm install
```

开发模式：

```bash
npm run dev
```

生产构建：

```bash
npm run build
```
